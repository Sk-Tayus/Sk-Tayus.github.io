## üìå Project Overview

**Flicker** is an online movie booking website that allows users to browse movies, select theatres, choose seats, add food & merchandise, and securely book tickets.

This project demonstrates a complete end-to-end flow of a modern movie booking system with an intuitive UI and structured navigation.

---

## üó∫Ô∏è Website Navigation Flow

1. **Homepage**

   * Search & explore movies
   * Popular movies display
   * Browse by categories (Action, Comedy, Drama, etc.)
   * Offers & promotions

2. **Movie Page**

   * Movie details (specifications, cast, trailers, reviews)
   * Option to book tickets

3. **Theatre Selection**

   * List of available theatres with showtimes
   * Dates & days of movie availability
   * Filters: movie language, tags, price range, special formats (IMAX, 3D, etc.)

4. **Seat Selection**

   * Interactive theatre seat layout
   * View available & booked seats
   * Seat selection before checkout

5. **Food & Merchandise (Optional)**

   * Add snacks, drinks, or movie merchandise to order
   * Can skip or proceed directly to payment

6. **Payment Gateway**

   * Review total bill (tickets + food/merch)
   * Secure payment processing

7. **Ticket Download**

   * Download ticket instantly after payment
   * Option to receive ticket via email
